The Road to Enlightenment

Back in 2003 (that's 8 years ago!) Dave Shea launched the CSS Zen Garden. It showcased what was possible with CSS-based designs, and that it gave you a complete control over the layout of a page.

At lot has happened since then, and today you can do almost anything with only CSS. CSS1K invites you to ..

So What is This About?

Do style sheets really have to be as heave as they are today? The average popular web site has 27k of CSS, but some use up to a megabyte of style sheets!

Can we acheive awesome things with just 1 K of CSS? I am confident that you can!

Rules

  1. Submissions may be up to 1 K (1024 bytes)
  2. Submissions must consist of only CSS
  3. Any and all external resources are strictly forbidden, this includes data URI:s, @font-face and @import
  4. The page does not have to look the same in every browser, but should degrade gracefully
  5. The submitted code is licensed under GPLv3

Download the sample HTML file and CSS file to work on a copy locally. Once you have completed your masterpiece (and please, don't submit half-finished work), send the CSS-file to submission@css1k.com.
