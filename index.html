The Road to Enlightenment

Back in 2003 (that's 8 years ago!) Dave Shea launched the CSS Zen Garden. It showcased what was possible with CSS-based designs, leading to an explosion of the use of CSS on the web.

At lot has happened since then, and today you can do almost anything with only CSS. CSS1K invites you to show that web developers are more inventive than ever, and that limitations can sparkle creativity.

So What is This About?

Do style sheets really have to be as heave as they are today? The average popular web site has 27 K of CSS, but some use up to a megabyte of style sheets!

Can we acheive awesome things with just 1 K of CSS? I am confident that you can!

Rules

  1. Submissions must consist of only CSS
  2. Submissions may be up to 1 K (1024 bytes) minified
  3. Vendor prefixes are not counted to the total number of bytes – submit your code unprefixed and we will add necesssary prefixes
  4. Any external resources and images, including data URI:s, @font-face and @import, are forbidden
  5. The page does not have to look the same in every browser, but graceful degradation is encouraged
  6. The submitted code is licensed under GPLv3

Download the sample HTML file and CSS file to work on a copy locally. Once you have completed your masterpiece (and please, don't submit half-finished work), send the CSS-file to submission@css1k.com.
